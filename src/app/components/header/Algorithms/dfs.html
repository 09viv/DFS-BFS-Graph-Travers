Depth-First Search (DFS)

DFS is an algorithm for traversing or searching tree or graph data structures. It starts at the root (or any arbitrary node) and explores as far as possible along each branch before backtracking.

How DFS works:

  • Start from the root node (or any arbitrary node).
  • Explore each branch deeply before moving to the next branch.
  • If a dead end is reached, backtrack and explore another branch.

Applications:

  • Cycle detection in graphs.
  • Solving puzzles (e.g., mazes).
  • Pathfinding in AI.
