Breadth-First Search (BFS)

BFS is an algorithm for traversing or searching tree or graph data structures. It starts at the root (or any arbitrary node) and explores the neighbor nodes at the present depth before moving on to nodes at the next depth level.

How BFS works:

  • Start from the root node (or any arbitrary node).
  • Explore all the neighbor nodes.
  • Move to the next level of nodes.
  • Repeat the process until all nodes are visited.

Applications:

  • Shortest path in unweighted graphs.
  • Web crawlers.
  • Peer-to-peer networks.
